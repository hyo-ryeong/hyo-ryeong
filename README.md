# 안녕하세요, 저는 프론트 개발자 [장효령]입니다! 👋

타인에게 망설임 없이 추천할 수 있는, 일상 속에 자연스럽게 스며드는 제품을 개발하는 것을 목표로 하고 있습니다. 좋은 제품을 만들기 위해 유저의 상황에 몰입하여 최선의 방향을 고민하고, 적극적으로 커뮤니케이션합니다.
